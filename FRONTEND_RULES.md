# Frontend Rules

Obligatorio: cargar y seguir el skill `ui-ux-pro-max` antes de escribir cualquier código frontend. Ejecutar `search.py --design-system` para obtener el sistema de diseño antes de empezar.

Avoid AI slop.

Always:
- premium SaaS aesthetic
- proper spacing
- subtle motion
- accessibility
- dark mode
- responsive
- reusable components
- usar CSS variables del tema shadcn (no colores hardcodeados)
- seguir el sistema de diseño generado por el skill
