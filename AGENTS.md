# Project Rules

Always load and follow SECURITY.md — applies to everything. Must load and follow the `owasp-security` skill when writing or reviewing security code.

When working on frontend (React, components, styles, UI), load FRONTEND_RULES.md and follow it strictly. Must load and use the `ui-ux-pro-max` skill (run `search.py --design-system`) before writing any UI code.

When working on backend (FastAPI, SQLAlchemy, models, endpoints, database), load BACKEND_RULES.md and follow it strictly. Must load and follow the `python-fastapi` skill.

Rules are mandatory unless explicitly discussed with the user.
